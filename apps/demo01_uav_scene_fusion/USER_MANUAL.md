# Demo-01: UAV Scene Fusion - 用户手册

## 1. 简介
本 Demo 展示了无人机遥测数据与 3D 场景的基础融合能力。通过接收 UDP 遥测数据，在基础的三维网格场景中实时更新无人机的位置和姿态，并绘制飞行轨迹。

## 2. 系统要求
- 操作系统：Ubuntu 20.04/22.04/24.04
- 依赖环境：Qt 5.15+ (包含 Qt3D)
- Python 3.8+（用于运行遥测数据模拟器）

## 3. 安装与运行
### 3.1 一键安装
在项目根目录或本目录下执行安装脚本：
```bash
./install.sh
```

### 3.2 运行程序
安装完成后，在 `build` 目录下运行可执行文件：
```bash
cd build
./Demo01UavSceneFusion
```

## 4. 操作说明
### 4.1 界面布局
- **左侧控制面板**：包含 UDP 控制、状态显示、无人机遥测数据显示和操作帮助。
- **右侧 3D 视图**：显示 3D 场景、无人机模型和飞行轨迹。

### 4.2 连接遥测数据
1. 点击左侧面板的 **Start UDP (14580)** 按钮，程序开始监听本地 14580 端口的 UDP 数据。
2. 运行遥测数据模拟器：
   ```bash
   python3 ../../../tools/telemetry_simulator/send_uav_udp.py
   ```
3. 连接成功后，UDP Status 将显示为绿色，并在 UAV Telemetry 面板中实时更新无人机的坐标、姿态和电量信息。

### 4.3 3D 视图操作
- **旋转视角**：在 3D 视图区域按住鼠标左键并拖动。
- **缩放视角**：滚动鼠标滚轮。
- **重置视角**：点击左侧面板的 **Reset View** 按钮，相机将恢复到初始位置。
- **清除轨迹**：点击左侧面板的 **Clear Trajectory** 按钮，将清空当前绘制的所有无人机飞行轨迹。

## 5. 常见问题
- **Q: 无法接收到 UDP 数据？**
  - A: 请检查端口 14580 是否被其他程序占用，并确保遥测数据模拟器正在运行且发送地址为 `127.0.0.1:14580`。
- **Q: 3D 场景显示异常或黑屏？**
  - A: 请检查显卡驱动是否正确安装，以及是否支持 OpenGL 3.3 及以上版本。
