# Demo-07: Game Event Evaluation - 用户手册

## 1. 简介
本 Demo 是整个项目的最终应用层展示。它在 3DGS 高保真渲染（Demo-05）的基础上，引入了游戏化的事件引擎、视觉特效系统和自动化评分机制。用户可以加载预定义的无人机巡检任务，回放飞行轨迹，并观察系统如何自动评估无人机是否到达了指定目标点或触发了特定区域。

## 2. 系统要求
- 操作系统：Ubuntu 20.04/22.04/24.04
- 依赖环境：Qt 5.15+ (包含 Qt3D)、OpenGL 4.3+
- 硬件要求：**必须配备独立显卡**（支持 Compute Shader），推荐显存 4GB 以上。

## 3. 安装与运行
### 3.1 一键安装
在项目根目录或本目录下执行安装脚本：
```bash
./install.sh
```

### 3.2 运行程序
安装完成后，在 `build` 目录下运行可执行文件：
```bash
cd build
./Demo07GameEventEvaluation
```

## 4. 操作说明
### 4.1 界面布局
- **左侧控制面板**：包含任务加载（Load Mission）、回放控制（Start/Pause/Reset）以及实时状态面板（得分、时间、电量、事件数）。
- **右侧 3D 视图**：显示 3DGS 场景、目标点标记、触发区域、无人机轨迹以及触发的视觉特效。
- **右侧评分统计**：回放结束时自动弹出，显示最终得分和完成情况。

### 4.2 加载任务与回放
1. 点击左侧面板的 **Load Mission** 按钮。
2. 选择位于 `sample_data/demo07_game/mission_config.json` 的任务配置文件。
3. 加载成功后，3D 视图中将出现橙色的目标椭圆和蓝色的触发区域线框。
4. 点击 **Start Replay** 开始回放轨迹。
5. 在回放过程中，当无人机轨迹点进入目标范围时，将触发相应的视觉特效（如发光、粒子爆发、虚拟爆炸）和右上角的文字弹窗（Label Popup），同时得分会实时增加。

### 4.3 视觉特效说明
- **Glow (发光)**：目标点发出柔和的光晕。
- **Pulse (脉冲)**：触发区域的边界线框产生高亮脉冲动画。
- **Particle (粒子)**：目标点喷发出大量的黄色粒子。
- **Explosion (虚拟爆炸)**：最高级的复合特效，包含中心闪光、多层环形波纹、橙红色喷发粒子、烟雾粒子和地面发光圆环。

## 5. 常见问题
- **Q: 回放速度太快或太慢？**
  - A: 默认回放速度由定时器控制（约 60FPS，每帧 16ms）。如果您的显卡渲染 3DGS 场景较慢，可能会导致实际回放时间长于预期。
- **Q: 为什么有些目标没有触发？**
  - A: 目标的触发基于 3D 空间距离计算（欧氏距离）。如果无人机的轨迹偏离了目标设定的触发半径（`radius`），则不会触发。您可以通过旋转视角（鼠标左键拖动）来观察轨迹与目标的实际空间关系。
