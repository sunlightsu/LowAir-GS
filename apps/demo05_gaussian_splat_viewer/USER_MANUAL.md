# Demo-05: Gaussian Splat Viewer - 用户手册

## 1. 简介
本 Demo 展示了如何在 Qt 框架中集成 3D Gaussian Splatting (3DGS) 渲染技术。3DGS 是一种新兴的辐射场渲染技术，能够以极高的帧率渲染出具有照片级真实感的 3D 场景。本程序提供了一个基础的 3DGS 模型加载和漫游工具。

## 2. 系统要求
- 操作系统：Ubuntu 20.04/22.04/24.04
- 依赖环境：Qt 5.15+ (包含 Qt3D)、OpenGL 4.3+ (支持 Compute Shader)
- 硬件要求：**必须配备独立显卡（NVIDIA 或 AMD）**，推荐显存 4GB 以上。

## 3. 安装与运行
### 3.1 一键安装
在项目根目录或本目录下执行安装脚本：
```bash
./install.sh
```

### 3.2 运行程序
安装完成后，在 `build` 目录下运行可执行文件：
```bash
cd build
./Demo05GaussianSplatViewer
```

## 4. 操作说明
### 4.1 界面布局
- **左侧控制面板**：包含模型加载控制、相机参数调节、渲染参数设置和操作帮助。
- **右侧 3D 视图**：使用 OpenGL 直接渲染的 3DGS 场景视图。

### 4.2 加载模型
1. 默认情况下，程序会尝试加载内置的示例模型。
2. 若要加载外部模型，点击左侧面板的 **Load PLY Model** 按钮。
3. 在弹出的文件选择对话框中，选择由 3DGS 训练工具生成的 `.ply` 格式模型文件。
4. 加载过程中，控制台会输出读取的 Splat 数量。加载完成后，模型将显示在 3D 视图中。

### 4.3 视图交互与参数调节
- **漫游控制**：
  - 按住鼠标左键拖动：旋转视角。
  - 滚动鼠标滚轮：缩放视角。
  - 按住鼠标右键拖动：平移视角。
- **参数调节**：
  - **Splat Scale**：调节高斯球的整体缩放比例，可用于填充模型空隙或观察底层结构。
  - **Camera FOV**：调节相机的垂直视场角。

## 5. 常见问题
- **Q: 运行程序时直接崩溃或黑屏？**
  - A: 本程序依赖 OpenGL 4.3 的 Compute Shader 进行高斯球的排序和渲染。如果您的显卡或驱动不支持 OpenGL 4.3，程序将无法运行。请在终端输入 `glxinfo | grep "OpenGL version"` 检查当前支持的版本。
- **Q: 模型加载很慢？**
  - A: 3DGS 的 `.ply` 文件通常非常大（数百 MB 到数 GB 不等），包含数百万个高斯球数据。加载时间取决于硬盘读取速度和 CPU 解析速度。请耐心等待。
