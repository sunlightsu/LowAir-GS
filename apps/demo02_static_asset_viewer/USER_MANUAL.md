# Demo-02: Static Asset Viewer - 用户手册

## 1. 简介
本 Demo 展示了静态 3D 资产（如 GLTF、OBJ 格式的 3D 模型）的加载、查看与交互功能。通过 Qt3D 框架，用户可以直观地查看无人机模型或场景资产，并进行多角度的观察。

## 2. 系统要求
- 操作系统：Ubuntu 20.04/22.04/24.04
- 依赖环境：Qt 5.15+ (包含 Qt3D)

## 3. 安装与运行
### 3.1 一键安装
在项目根目录或本目录下执行安装脚本：
```bash
./install.sh
```

### 3.2 运行程序
安装完成后，在 `build` 目录下运行可执行文件：
```bash
cd build
./Demo02StaticAssetViewer
```

## 4. 操作说明
### 4.1 界面布局
- **左侧控制面板**：包含模型加载控制、材质/光照设置和操作帮助。
- **右侧 3D 视图**：显示加载的 3D 模型资产。

### 4.2 加载模型
1. 默认情况下，程序会加载内置的简单无人机模型。
2. 若要加载外部模型，点击左侧面板的 **Load Model** 按钮。
3. 在弹出的文件选择对话框中，选择支持的 3D 模型文件（如 `.obj`, `.gltf` 等）。
4. 模型加载成功后，将自动在 3D 视图中居中显示。

### 4.3 视图交互
- **旋转视角**：在 3D 视图区域按住鼠标左键并拖动，可全方位旋转观察模型。
- **缩放视角**：滚动鼠标滚轮，可放大或缩小模型。
- **平移视角**：按住鼠标右键（或 Shift + 左键）并拖动，可平移相机位置。
- **重置视角**：点击左侧面板的 **Reset View** 按钮，相机将恢复到默认观察角度和距离。

## 5. 常见问题
- **Q: 模型加载后显示为黑色或没有纹理？**
  - A: 请确保模型文件及其关联的材质文件（如 `.mtl`）和纹理图片在同一目录下，且路径中不包含特殊字符。
- **Q: 加载大型模型时程序卡顿？**
  - A: 静态资产查看器主要用于轻量级资产验证，对于面数过高（如数百万面）的模型，可能会出现性能瓶颈，建议对模型进行轻量化处理。
