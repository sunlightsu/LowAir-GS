# Demo-06 Sample Data

本目录用于存放 Demo-06（双源场景融合）的示例数据和对齐配置文件。

## 文件说明

| 文件名 | 类型 | 说明 |
|---|---|---|
| `building_mesh.obj` | 摄影测量模型 | 简化的立方体建筑物网格模型（10 顶点，16 三角面） |
| `building_gaussians.ply` | 3DGS 点云 | 环绕建筑物的植被与环境高斯点云（1300 点） |
| `fusion_config.json` | 融合配置文件 | 记录两个资产之间的相对变换参数（平移、旋转、缩放）及渲染状态 |

## 生成方式

本目录下的数据由 Python 脚本自动生成。如果文件缺失或需要重新生成，请在仓库根目录执行：

```bash
python3 tools/gaussian_generators/generate_demo06_assets.py
```

## 注意事项

- 本目录仅用于存放**体积极小**的程序生成测试数据。
- 真实的摄影测量模型（数 GB）和真实 3DGS 场景（数 GB）**严禁**提交到 Git 仓库。
- 在进行真实数据测试时，建议将数据存放在仓库外的独立目录中，通过绝对路径加载。
- **数据声明**：本目录中的所有文件均为教学模拟双源数据，不是真实同源测绘结果。Mesh 和 Gaussian 分别来自两个独立的程序生成过程，仅用于验证双源融合渲染流程的正确性，不代表真实场景的摄影测量精度。
